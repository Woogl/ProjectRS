# ProjectRS
Party-based ARPG
Hello
프로젝트 RS에서 사용되는 용어의 정리
방어력, 보호력, 방어율, 합격기, 협동기, 합체기 같은 혼동이 오는 단어를 정리, 미스 커뮤니케이션을 줄인다.
기조
내부 변수명은 영어로 사용된다. 
구별을 위해 게임 UI상에서 표현하는 용어는 한글로 사용한다. 
HP X 생명력 O

생명력 
쿨타임
링크 스킬: 교체기, 합격기라고 불리던 기존 교체기를 표현하는 용어입니다. 대신 패링해주기, 지켜주기 등 다른 캐릭터와 상호작용을 하나, 교체하지 않거나 캐릭터 단독으로 쓰는 상황을 생각하여 링크 스킬로 명명합니다.
전투 스킬: 쿨타임으로 관리하는 캐릭터마다 가진 스킬들
궁극기
궁극기 게이지
력, 계수: 캐릭터의 능력치는 력 스킬의 능력치는 계수로 표현합니다.
공격력
스킬 계수
충격력: 캐릭터에게 매핑된 그로기 게이지 쌓는 능력치
충격 계수: 스킬에 매핑된 그로기 게이지 쌓는 능력치
그로기 수치
치명타 확률
치명타 피해
